# 前端磨砂玻璃简约化改造（2026-06-10）

## 目标

把现有和纸风格前端往 iOS 26 liquid glass 方向调整，适配 Mac，减少视觉元素；修复走查中发现的 bug。不改布局结构，不拆分 App.vue。

## 已确认的约束

- 字体保留宋体/明朝体（用户明确要求，不换系统字体）。
- 保留现有暖色调与赤陶色（terracotta）强调色。
- 布局、功能、交互流程不变。

## 改造内容

1. **玻璃材质**：主要卡片/面板加 `backdrop-filter: blur + saturate`，表面色改为更通透的半透明，配高光细边框（浅色与深色模式分别调）。
2. **环境背景**：body 加柔和环境渐变（浅色：暖米白多点渐变；深色：深暖灰 + 微光晕），让玻璃面板有可透内容。
3. **减少视觉元素**：
   - 「线程摘要」无内容时不渲染空面板；
   - 顶部 LLM provider 胶囊视觉弱化；
   - 例句绿/粉/黄三色荧光高亮统一为单一柔和强调色；
   - 阴影整体减轻。
4. **Bug 修复**：深色模式选中 tab 白底白字（`.app-dark .mode-switch button.active` 设了 `#f8fafc` 背景但文字仍为浅色）。

## 验证方式

- `npm run build` 通过；
- preview 截图核对浅色/深色模式下四个页面（词典查询、单词复习、说明、变形道场）；
- 深色模式选中 tab 文字可读；
- 道场答题、记忆评级流程回归正常。
